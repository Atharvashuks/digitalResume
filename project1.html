Blogries, Build a full stack MERN web-application for blogging

  • Live Demo
  • Source code

Developed a full stack MERN web-application. People can read other peoples blogs on the website. Users can login to their accounts and can create, update, delete their posts. Full security for the users posts is provided.

This website is developed using React JS as the main front-end framework, and Node JS is used for building backend structure and apis. MongoDB atlas is used as database. State management is done by the implementation of redux.

  • Google auth is also enabled
  • Users need to log in for creating updating and liking posts. However users can read other posts on website without loging in.
  • Some very usefull features like pagination is also included in the project.
